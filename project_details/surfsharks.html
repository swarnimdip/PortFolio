Technical Details

Surf Sharks is a endless surfing game, in which surfer can change buy and update skins according to experience level and have to surf untill the lives end bu jumping or dodging the emenies.

Project is written in C#, and firebase tools used to creates the game features

  • Created the endless gameplay with emenies along with underwater mode
  • Handled all animation from spine
  • Integrated firebase tools like - Authentication, Firestore, Storage, Remote config, Messanging and Hosting
  • Added and configured IAP and In-App-Update for both Android and IOS
  • Handled the skin updation process from admin side without updating game
  • Published the game for client on both Playstore and Appstore
